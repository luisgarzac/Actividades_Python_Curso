# **Ejercicios de práctica**

Los siguientes ejercicios te permitirán poner en práctica lo aprendido en el curso de Introducción a la programación en Python. Encontrarás en esta sección las actividades que tendrás que realizar en formato de enunciados para diferentes temas en Python y las soluciones las encontrarás en la carpeta de "***ejercicios resueltos***"


---

> ### **Ejercicios de control de flujo con 'if':**

1.   En un programa Python, pide al usuario que ingrese un número. Si el número es positivo, imprime "El número es positivo". Si es negativo, imprime "El número es negativo". Si es cero, imprime "El número es cero".

2.   Escribe un programa Python que pida al usuario una contraseña. Si la contraseña es "contraseña123", imprime "¡Bienvenido!". Si la contraseña es incorrecta, imprime "Contraseña incorrecta, vuelve a intentarlo".

3.   Escribe un programa Python que pida al usuario que ingrese un número y determine si es par o impar.

4.   Escribe un programa Python que pida al usuario su edad y determine si es mayor o menor de edad (mayor o igual a 18 años).

---

> ### **Ejercicios de bucles 'for':**

1.   Escribe un programa Python que imprima los primeros 10 números pares.

2.   Escribe un programa Python que pida al usuario una cadena de texto e imprima cada letra de la cadena en una línea separada.

3.   Escribe un programa Python que pida al usuario un número y imprima la tabla de multiplicar de ese número hasta 10.

---
> ### **Ejercicios de arreglos 'arrays':**

1.   Escribe un programa Python que cree un arreglo (array) de números del 1 al 10 y luego imprima los elementos del arreglo en orden inverso.

2.   Escribe un programa Python que pida al usuario que ingrese 5 números y luego imprima el promedio de los números.

3.   Escribe un programa Python que cree un arreglo (array) de números pares del 2 al 20 y luego imprima los elementos del arreglo en orden ascendente.

4.   Escribe un programa Python que pida al usuario que ingrese 5 números y luego imprima el número más grande de los números.

---
> ### **Ejercicios de 'listas':**

1.   Escribe un programa Python que pida al usuario que ingrese números hasta que ingrese -1. Luego, imprime los números ingresados en orden inverso.

2.   Escribe un programa Python que pida al usuario que ingrese números hasta que ingrese -1. Luego, imprime el número más grande de la lista.

3.   Escribe un programa Python que pida al usuario que ingrese números hasta que ingrese 0. Luego, imprime los números ingresados en orden ascendente.

4.   Escribe un programa Python que pida al usuario que ingrese números hasta que ingrese 0. Luego, imprime la suma de los números ingresados.

---
> ### **Ejercicios de 'Numpy':**

1.   Escribe un programa Python que cree un arreglo (array) de numpy de números aleatorios entre 0 y 1 y luego calcule la media y la desviación estándar de los elementos del arreglo.

2.   Escribe un programa Python que cree dos arreglos (arrays) de numpy de números aleatorios entre 0 y 1 y luego calcule su producto punto (dot product).

3.   Escribe un programa Python que cree un arreglo (array) de numpy de números enteros del 1 al 10 y luego multiplique cada elemento del arreglo por 2.

---
> ### **Ejercicios de 'Scipy':**

1.   Escribe un programa Python que calcule la raíz cuadrada de un número utilizando la función sqrt de la librería scipy.

2.   Escribe un programa Python que calcule la integral de la función seno de x desde 0 hasta pi utilizando la función quad de la librería scipy.

3.   Escribe un programa Python que resuelva el sistema de ecuaciones lineales 2x + y = 5 y x + y = 3 utilizando la función solve de la librería scipy.

---
> ### **Ejercicios de 'Diccionarios':**

1.   Escribe un programa Python que cree un diccionario que contenga las edades de varios amigos y luego imprima la edad del amigo más joven.

2.   Escribe un programa Python que cree un diccionario que contenga las notas de varios estudiantes y luego calcule el promedio de las notas.

3.   Escribe un programa Python que cree dos diccionarios que contengan la cantidad de dinero que tienen varias personas en su cuenta corriente y luego combine los diccionarios en uno solo.
